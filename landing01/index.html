Font-awesome icons

This landing-page use Font-awesome icons. While there are numerous ways to add icons to a website, the most popular way is to use Font Awesome. Font-awesome icons can be formatted using css properties, set their color, size, shadow, and more...

Slider

A slider is a special element of a website; it is a block of a certain width. Its main feature is the elements that change in a manual or automatic mode - pictures, text and links.

Responsive for devices

Responsive websites are those crafted to use W3C CSS3 media queries with fluid grids and commonly also fluid images, to adapt the layout to the viewing environment. The idea of responsive web design is to make websites that fit multiple screen sizes. For the designer and developer that means they only have to work on one version of the website instead of designing one for desktop computers and another for mobile devices.
